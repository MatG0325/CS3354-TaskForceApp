# TaskForce
TaskForce is a tool used for scheduling and task management. It provides a plethora of functionality, including multi-factor authentication, managerial oversight, and AI-empowered schedule optimization. This project utilizes React.js (frontend), Node.js (Backend), PostgreSQL (data), and AWS (server hosting).

This project was created for Dr. Smith's 3354 Software Engineering course.
